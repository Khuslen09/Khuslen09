## 🚀 Featured Projects

### 📍 TRACEN — Flutter Location Life-Log App

> 나의 하루 이동을 지도 위에 기록하는 라이프로그 앱  
> A Flutter app that visualizes your daily movement and life on a map

```
Flutter · Firebase · Google Maps · Gemini API · SQLite
```

| 기능 | Feature |
|---|---|
| 🗺 CustomPainter + BlendMode로 스크래치 맵 구현 | Scratch map visualization with CustomPainter & BlendMode |
| 🏃 GPS 기반 러닝 추적 (거리 · 페이스 · 경로 히스토리) | Live running tracking with GPS, distance, pace & route history |
| 📌 사진 · 메모 · 카테고리 핀 + Firebase 동기화 | Memory pins with photos, notes, categories & Firebase sync |
| 🤖 Google Places + Gemini API 기반 AI 장소 추천 | AI place recommendations via Google Places + Gemini |
| 💾 SQLite 로컬 우선 아키텍처 + Firestore 동기화 | Local-first architecture with SQLite & Firestore sync |

> 🎓 현재 캡스톤 디자인 프로젝트로 고도화 진행 중 | Currently improving as capstone design project
